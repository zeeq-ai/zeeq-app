# Zeeq Testing Facilities

Common testing utilities, base classes, and fixtures for Zeeq unit and integration tests. This project is intended to be referenced by test projects within the Zeeq solution to provide shared testing infrastructure.

## EntityGraphBuilder

Use `EntityGraphBuilder` for terse backend test seed graphs with realistic
Zeeq users, organizations, teams, memberships, and pending invitations.

See [EntityGraph/README.md](EntityGraph/README.md).
